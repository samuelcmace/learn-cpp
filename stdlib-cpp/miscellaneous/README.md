# cmdcli
A CMake compiled launcher for espeak and cowsay with more features.

## Features
 - xcowsay support for graphical cowsay
 - "exit" command exits application

## Build Directions
To build on Linux, install cmake, make, and g++

### Then Complie

```mkdir build```

```cd build```

```cmake ..```

```make```

### Then Enjoy :smiley:
