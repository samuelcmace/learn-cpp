## Build Requirements:
- CMake

## Build instructions for Ubuntu:

### Download the repository:
```bash
git clone https://github.com/Sammylandler/math.git
```

### Install the necessary build tools:
```bash
sudo apt install cmake g++ -y
```

### Compile using CMake:
```bash
mkdir build
cd build
cmake ..
make
```

### Then, run the desired binary programs.
